Создано на базе библиотеки openCV для Python.

Данный проект создан для распознавания различных объектов
на фото. Разрабатывался для распознавания лиц, однако
может распознавать и другие объекты при замене xml-файла
и настройке гиперпараметров.

Необходимые для работы файлы:
1. Каталог "images" с фотографиями;
2. Файл .xml для создания ML-модели.
